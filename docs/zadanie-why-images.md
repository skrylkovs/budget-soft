# Задание: картинки блока «Почему выбирают BUDGET SOFT» для топ-5 страниц услуг

Дата: 12.07.2026. Статус: к генерации.

## Контекст

На всех страницах услуг в блоке «Почему … выбирают BUDGET SOFT» сейчас один общий
комплект из 4 иллюстраций `images/why/{projects,mvp,price,expertise}.png`
(сгенерирован по текстам страницы ERP). Исключения:

- `/uslugi/telegram-mini-apps/` — уже получила свой комплект `images/why/tg-*.svg`;
- `/uslugi/razrabotka-erp-sistem/` — фактически закрыта текущим глобальным
  комплектом (он рисовался по её карточкам).

Задача — сгенерировать **свои комплекты по 4 картинки** для топ-5 страниц ниже,
чтобы иллюстрации соответствовали текстам карточек конкретной услуги.

Как показывается картинка: слева от текста карточки, ширина **115px** (высота по
пропорции), фон карточки белый. Главный элемент должен читаться в этом размере.

## Общие требования к генерации

Базовый шаблон промпта — проверенный (им сгенерирован текущий комплект):

```
нарисуй 4 картинки размером 100×100 с прозрачным фоном
цветовая гамма - темная
стиль - минималистичный, cyberpunk
```

Дополнения к шаблону (важны для серии):

- в каждый промпт добавлена строка «тематика страницы …» — чтобы картинки
  разных страниц отличались, ведь 2 из 4 карточек («60+ проектов за 17 лет»,
  «До 4× дешевле») повторяются на всех страницах;
- надписи внутри картинок — только крупные и короткие («60+», «4×», «MVP»);
  кириллицу генераторы часто ломают — после генерации проверить каждую букву,
  при артефактах перегенерировать или просить вариант без текста;
- фон строго прозрачный (картинка лежит на белой карточке);
- один доминирующий объект на картинку — при 115px мелкие детали пропадают.
