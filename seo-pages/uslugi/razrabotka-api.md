## 3. FAQ — Частые вопросы о разработке API

**1. Сколько стоит разработка API?**
От 200 000 ₽ [≈ согласовать] за REST API с документацией и аутентификацией; цена зависит от числа эндпоинтов, протокола и требований к нагрузке. Смету считаем бесплатно после разбора интеграций.

**2. REST, GraphQL или gRPC — что выбрать?**
REST — универсальный выбор для интеграций и партнёров, GraphQL — для гибких запросов фронтенда, gRPC — для быстрого обмена между сервисами. Подбираем протокол под задачу, часто совмещаем несколько.

**3. Будет ли документация к API?**
Да: контракт описываем в OpenAPI (Swagger), потребители подключаются по спецификации. Документация ведётся вместе с кодом и обновляется при версионировании.

**4. Как защищён доступ к API?**
Аутентификация через OAuth 2.0, JWT или API-ключи, разграничение прав, rate limiting и логирование через API Gateway — доступ контролируется, лимиты защищают от перегрузки и злоупотреблений.

**5. Можно развернуть API на наших серверах?**
Да: разворачиваем on-premise или в вашем облаке, замещаем закрытые зарубежные API. Свой код и права остаются у вас.
