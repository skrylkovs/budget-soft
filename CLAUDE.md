# Правила проекта BUDGET SOFT (prototype-main)

## URL существующих страниц неизменны
- Реестр всех URL — [urls.json](urls.json). Это источник истины по адресам страниц.
- **Никогда не меняй адрес существующей страницы**: не переименовывай и не удаляй её директорию,
  не трогай путь `<dir>/index.html`. На эти URL могут вести внешние ссылки, поисковая индексация и закладки.
- Менять можно только **содержимое** страницы, а не её адрес.
- При добавлении **новой** страницы — создай `<новый-dir>/index.html` и добавь запись в `urls.json`,
  существующие записи не трогай.
- Если задача требует изменить адрес существующей страницы — сначала предупреди и спроси подтверждение,
  и предложи редирект со старого URL вместо простого переименования.

## Структура URL
Сайт статический, «красивые» URL: каждая страница — `<dir>/index.html`, отдаётся как `/<dir>/`.
