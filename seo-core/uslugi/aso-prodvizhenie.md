## Рекомендации по странице

1. **Title / H1:** H1 — «ASO-продвижение мобильных приложений» (P0-связка «aso» +
   широкий запрос). Title — добавить сторы (App Store, Google Play, RuStore) и цену «от»
   после согласования. Description — сервисные формулировки + цена + сроки первых
   результатов.
2. **Объясняющий слой:** блок «что такое ASO и зачем оно» обязателен — выдача широкого
   запроса статейная, Алиса-виджет отвечает «как работает ASO».
3. **Цена:** пакеты «аудит / оптимизация / сопровождение» с цифрами — у 5 из 6
   разобранных конкурентов цены есть (вилка 40–120 тыс. ₽); цифры согласовать.
4. **Сторы:** карточки App Store / Google Play / RuStore и альтсторы — закрывают
   подкластер 3; RuStore — пустая ниша (почти никто не выделяет).
5. **ASO vs реклама:** сравнение органического ASO и платного трафика таблицей +
   перелинковка на контекст/SMM.
6. **Этапы:** аудит → семантика → метаданные и визуал → релиз → итерации; сроки первых
   результатов (2–4 недели — рыночный ориентир appfox) согласовать.
7. **FAQ:** 8–10 вопросов — «сколько стоит», «когда результаты», «что такое ASO», «можно
   ли без рекламы», «какие сторы», «что с играми», «как меряете эффект», «что нужно от нас».
8. **Перелинковка:** mobilnaya-razrabotka (разработка), kontekstnaya-reklama / smm
   (платные каналы), telegram-mini-apps при упоминании экосистемы.
9. **Чего не делать:** не таргетить «aso» соло (омоним); не обещать топ и фичеринг; не
   выдумывать кейсы с цифрами установок.
